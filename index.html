About Me

Hi, my name is Zachary Muranaka and I'm a computer science student and member of the Aletheia club at Weber State University. I love to code and am proficient with the languages HTML, CSS, JavaScript, PHP, C++, Java, Python, and SQL. I am also proficient with the command-line utilities Windows Command Prompt and Linux Bash Shell, as well as the version-control system Git. When I'm not working on schoolwork, you can probably find me reading or working on a new coding project.

Hypergeometric Calculator

Responsive web application that solves hypergeometric distributions, which describe the probability to get a number of desired successes out of a number of draws from a population without replacement. I built this mainly with JavaScript, which performs the calculations and writes them to the HTML.

Hack Game

If you've ever seen a movie or TV show that portrays a "hacker", they were probably furiously typing cryptic green text into a black terminal. This is a simple game to simulate that experience, by turning almost any key the user types into a line of green "hacker code". I built this with Python and the framework Pygame Zero.
