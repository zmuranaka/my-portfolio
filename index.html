About Me

Hi! My name is Zachary Muranaka and I'm a computer science student and member of the Aletheia club at Weber State University. I love to code and am proficient with several programming languages like HTML, CSS, JavaScript, C++, Java, and SQL. I know a bit of Python, PHP, Linux Terminal, Windows Command Prompt, and Git as well. When I'm not working on schoolwork, you can probably find me helping my sister with some of her math or working on a new coding project.
